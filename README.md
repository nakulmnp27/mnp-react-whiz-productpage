# mnp-react-whiz-productpage

## Whizlabs Product Page – React Version
### Overview

This project is a React-based dynamic implementation of the Whizlabs Product Details Page for the AWS Certified Cloud Practitioner (CLF-C02) course.

The original static HTML/CSS page has been converted into reusable React components and all course information is now fetched from a backend API / database instead of being hardcoded.

### Tech Stack

React

JavaScript

HTML5 

CSS3 or Bootstrap

Node.js & Express (backend)

Database (PostgreSQL)

### Features

Component-based UI

Dynamic data from API

Responsive layout
